## &#128039; Descripción 
El proyecto consiste en realizar el landing page de la agencia **Viajes Chile** utilizando los conocimientos adquiridos a través de las distintas unidades del módulo.

## &#128507; Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap 5.3.3
- FontAwesome 6
- Google Fonts
